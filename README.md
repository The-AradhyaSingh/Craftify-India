# 🛍️ Craftify India

A modern, responsive website for showcasing and selling handcrafted
products. Built using HTML, CSS, and JavaScript, this project highlights
local artisans and eco-friendly items.

------------------------------------------------------------------------

## ✨ Features

-   🧼 Handcrafted product categories (soap, jewelry, candles,
    furniture, etc.)
-   🎨 Clean UI with gradients and animations
-   📱 Responsive design (mobile-friendly)
-   🛒 Basic cart system (JavaScript)
-   ⚡ Smooth scrolling navigation
-   🌿 Focus on eco-friendly products

------------------------------------------------------------------------

## 🚀 Getting Started

### 1. Download / Clone
git clone https://github.com/The-AradhySingh/craftify-india.git

### 2. Run Project

-   Open folder\
-   Double-click index.html\
-   Or use Live Server
