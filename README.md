# 🛍️ Craftify India – Handcrafted Products Website

Craftify India is a modern, responsive e-commerce website designed to showcase and sell handmade products. It promotes local artisans and eco-friendly goods with a clean and attractive user interface.

---

## 📌 Features

- 🧼 Multiple handcrafted product categories
- 🎨 Modern UI with animations
- 📱 Fully responsive design
- 🛒 Simple cart functionality (JavaScript-based)
- 🌿 Eco-friendly product focus
- ⚡ Smooth scrolling navigation
